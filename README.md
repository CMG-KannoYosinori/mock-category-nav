# mock-category-nav

モラタメアプリの商品一覧ページ（タメせる／メディタメ）のカテゴリナビ・タブ UI をモックアップするリポジトリです。

---

## swiper の実装参考

https://au.moratame.net/?sitemove=coupon-top_ichioshi2

---

## ファイルの役割

### HTML・CSS（`s/app_published/`）

| ファイル                              | 役割                                                                                                                                             |
| ------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| **list_tamesu--original.html**        | モックアップ前の元ファイル。Slick による「タメせる／メディタメ」2タブのみ。カテゴリ（総合・食品・ドリンク…）はなし。                             |
| **list_tamesu--swiper.html**          | アップデート後の想定形。タメせる／メディタメはタブで切り替え、その内側に Swiper でカテゴリナビ（総合・食品・ドリンク…）＋ カテゴリ別カード一覧。 |
| **list_tamesu--swiper.css**           | swiper 版専用スタイル（タブ・Swiper・グローバルナビ等）。`list_tamesu--swiper.html` から相対パスで読み込み。                                     |
| **list*tamesu--swiper*運用注意点.md** | カテゴリナビ（Swiper）の運用注意。メニューとコンテンツの数・順序、HTML 構造、トラブル時の確認項目。実装時は必ず参照。                            |

### ドキュメント（リポジトリ直下）

| ファイル                           | 役割                                                                                                                        |
| ---------------------------------- | --------------------------------------------------------------------------------------------------------------------------- |
| **REFACTORING.md**                 | Slick カルーセルからタブナビゲーションへのリファクタリング内容。タブの HTML/CSS/JS、sessionStorage、GA 等の変更履歴。       |
| **HANDOVER_ORIGINAL_TO_SWIPER.md** | 実装チーム向け引き継ぎ資料。`list_tamesu--original.html` を `list_tamesu--swiper.html` と同様の UI にアップデートする手順。 |

### スクリプト（`scripts/`）

モックアップ作成時に使用したスクリプト。

| ファイル                     | 役割                                                                                                    |
| ---------------------------- | ------------------------------------------------------------------------------------------------------- |
| **categorize_cards.py**      | カードをキーワードでカテゴリ分類するスクリプト。カテゴリ振り分け時に参照。                              |
| **apply_category_slides.py** | `category_slides_output.txt` を用いて、食品〜クーポンなどのスライド HTML を一括差し替えするスクリプト。 |

### その他

| ファイル               | 役割                                                                               |
| ---------------------- | ---------------------------------------------------------------------------------- |
| **purgecss.config.js** | `list_tamesu--swiper.css` を本番配布する際の PurgeCSS 設定。未使用スタイル削除用。 |
